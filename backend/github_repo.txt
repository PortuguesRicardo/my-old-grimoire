for the Assessor:
1. Clone the repository.
2. Navigate to the backend folder and install dependencies with:
   npm install (--legacy-peer-deps)


3. Create a .env file in the backend folder with:
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
4. Start backend with:
npm start  or npm run dev
5. Start frontend with:
npm start in the frontend folder.

The application is fully functional and matches the project specifications.
